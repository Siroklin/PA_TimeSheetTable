# Руководство пользователя — «График работы сотрудников»

## 1. Вход в систему

При открытии приложения показывается стартовый экран с логотипом и названием.

1. Нажмите кнопку **«Войти»**.
2. Введите **логин** и **пароль**, нажмите **«Войти»**.

Стандартный администраторский доступ при первой установке: логин `admin`, пароль `admin`.
**Сразу после первого входа смените пароль администратора** (раздел [6.2](#62-управление-пользователями)).

Сессия действует, пока открыта вкладка браузера — после закрытия вкладки нужно войти снова.
Чтобы выйти вручную: **Сервис → Выйти**.

---

## 2. Главный экран

После входа открывается таблица графика работы за текущий месяц для одного из доступных вам отделов.

Сверху расположена панель фильтров:

| Фильтр | Назначение |
|---|---|
| Переключатель месяца/года (стрелки `‹` `›` и выпадающие списки) | Период, за который показывается график |
| **Отдел** | Какой отдел сейчас показан (доступны только те отделы, которые вам назначены) |
| **Должность** | Показать только сотрудников выбранной должности |
| **Смена** | Показать только тех, у кого есть смены «День» или «Ночь» в этом месяце |

Справа сверху — кнопка **«Скачать Excel»**: выгружает текущий отдел/месяц в файл .xlsx. Доступна всем, включая пользователей с ролью «Просмотр».

Кнопка **«Сервис»** справа от фильтров открывает меню действий — какие пункты видны, зависит от вашей роли (см. раздел [5](#5-роли-и-права-доступа)).

### 2.1 Таблица графика

- Строки — сотрудники (ФИО и должность), столбцы — дни месяца, каждый день разбит на колонки **Д** (день) и **Н** (ночь).
- Возле имени сотрудника — значок **Д/Н**, показывающий его текущую смену, кнопка **«Граф.»** (быстрое заполнение графика по шаблону) и кнопка **«×»** (удалить сотрудника).
- Цвет ячейки соответствует статусу (см. легенду под таблицей):

| Обозначение | Статус | Цвет |
|---|---|---|
| Р | Работает | зелёный |
| В | Выходной | жёлтый |
| О | Отпуск | синий |
| Б | Больничный | красный |
| С | Отсыпной | фиолетовый |
| ДО | Отпуск за свой счёт | оранжевый |
| (пусто) | Не указано | серый/белый |
| Ночь | — | всегда серая, кроме отметки **Р** |

- Красная точка в ячейке — есть комментарий (виден при наведении или в окне редактирования).

## 3. Редактирование графика (роль «Редактирование» и администраторы)

### 3.1 Изменить одну ячейку

1. Кликните по ячейке Д или Н нужного дня для нужного сотрудника.
2. В открывшемся окне выберите статус (Р, В, О, Б, С, ДО) и, при необходимости, впишите комментарий.
3. Нажмите **«Сохранить»**.

### 3.2 Быстрое заполнение графика («Граф.»)

Кнопка **«Граф.»** возле сотрудника открывает заполнение графика по шаблону на весь месяц:

| Шаблон | Описание |
|---|---|
| **2×2** | 2 рабочих дня / 2 выходных, скользящий цикл |
| **ДНОВ** | День → Ночь → Отсыпной → Выходной (4-дневный цикл, занимает обе колонки — Д и Н) |
| **5-0** | Пн–Пт работа, Сб–Вс выходной |
| **6-1** | Пн–Сб работа, Вс выходной |

Для **2×2** и **ДНОВ** нужно указать дату начала цикла (первый рабочий день) — это позволяет циклу «продолжаться» корректно при копировании на следующие месяцы (см. 3.3). Для **2×2** и **5-0**/**6-1** дополнительно выбирается смена сотрудника — дневная или ночная.

Внизу окна — предпросмотр первых 14 дней. Нажмите **«Применить»**, чтобы перезаписать график сотрудника на весь месяц по шаблону (старые значения за этот месяц будут стёрты).

### 3.3 Копировать расписание

**Сервис → Копировать расписание** — копирует график **всех сотрудников текущего отдела** с указанного месяца на другой месяц/год.

- Если у сотрудника график был внесён через шаблон («Граф.»), при копировании скользящие циклы (2×2, ДНОВ) **продолжаются без разрыва** (а не начинаются заново).
- Если шаблон не использовался — данные копируются как есть, день-в-день.
- Существующий график в целевом месяце при копировании **будет перезаписан**.

## 4. Сотрудники и справочники (роль «Редактирование» и администраторы)

### 4.1 Добавить одного сотрудника

**Сервис → Добавить сотрудника** → заполните код, ФИО, должность (выбирается из справочника должностей этого отдела) → **«Добавить»**.

### 4.2 Загрузить список сотрудников (CSV)

**Сервис → Загрузить сотрудников**.

Формат CSV (через запятую, без заголовка): `код,отдел,ФИО,должность`, например:
```
001,Цех №1,Иванов Иван Иванович,Оператор
002,Склад,Петрова Мария Сергеевна,Кладовщик
```
- Отдел в каждой строке должен совпадать с одним из существующих названий отделов.
- Можно вставить текст в поле или загрузить .csv/.txt файл.
- Нажмите **«Проверить»** — система покажет ошибки или таблицу предпросмотра.
- Нажмите **«Загрузить»** — сотрудники, у которых код+отдел совпадает с уже существующими, пропускаются (без дублей), остальные добавляются.

### 4.3 Удалить сотрудника

Кнопка **«×»** в строке сотрудника в таблице графика. Действие необратимо — удаляются сам сотрудник и весь его график.

### 4.4 Управление должностями

**Сервис → Управление должностями** — справочник должностей конкретного отдела (используется в выпадающем списке при добавлении сотрудника и в фильтре «Должность»).

- Переключить отдел можно прямо в этом окне.
- Добавить новую должность — введите название и нажмите **«Добавить»**.
- Удалить должность — кнопка **«×»** рядом с ней.

## 5. Роли и права доступа

Каждому пользователю (кроме администраторов) назначается:
- **список отделов**, которые он видит — в фильтре «Отдел» доступны только они;
- **роль**:

| Роль | Что доступно |
|---|---|
| **Редактирование** | Полный доступ в рамках своих отделов: просмотр, редактирование графика, добавление/удаление сотрудников, управление должностями, копирование расписания, скачивание Excel |
| **Просмотр** | Только просмотр графика по своим отделам и скачивание Excel. Пункты редактирования в меню «Сервис» не отображаются, клик по ячейке графика ничего не открывает |

**Администратор** не ограничен ролью и отделами — видит и может редактировать всё, плюс имеет доступ к управлению отделами и пользователями.

## 6. Раздел администратора

Пункты ниже видны только пользователям с правами администратора.

### 6.1 Управление отделами

**Сервис → Управление отделами**:
- Список всех отделов в системе.
- Добавить отдел — ввести название и нажать **«Добавить»**.
- Удалить отдел — кнопка **«×»**. Появится подтверждение: *«Вы точно хотите удалить отдел "X"? Все данные по отделу будут удалены.»* При подтверждении безвозвратно удаляются: все сотрудники отдела, их график, шаблоны заполнения, должности отдела и доступ пользователей к этому отделу.

### 6.2 Управление пользователями

**Сервис → Управление пользователями**:
- Список пользователей с указанием логина, имени, роли и доступных отделов.
- **Добавить пользователя**: имя, почта, логин, пароль, флажок «Администратор», роль («Редактирование»/«Просмотр»), отметить чекбоксами доступные отделы.
- **Редактировать** (значок ✎): те же поля; если оставить пароль пустым — он не меняется.
- **Удалить** (значок ×): нельзя удалить самого себя, чтобы не потерять доступ.

Если включён флажок «Администратор» — роль и список отделов не имеют значения, у пользователя полный доступ ко всему.

## 7. Частые вопросы

**Не вижу нужный отдел в фильтре.**
Значит он вам не назначен — обратитесь к администратору, чтобы он добавил отдел в список доступных в настройках вашего пользователя.

**Не вижу кнопок добавления сотрудников/«Граф.»/пункты в меню «Сервис».**
У вашей учётной записи роль «Просмотр» — для редактирования обратитесь к администратору.

**Случайно перезаписал график при копировании или заполнении по шаблону.**
Действие необратимо откатить через интерфейс нельзя — данные за месяц нужно будет ввести заново. Будьте внимательны при выборе целевого месяца перед копированием.

**Забыли пароль.**
Только администратор может сбросить пароль — через «Управление пользователями» → редактирование пользователя → ввести новый пароль.
